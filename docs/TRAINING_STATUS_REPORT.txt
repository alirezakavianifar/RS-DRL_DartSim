SOLUTION:
---------
DARTSim needs to be started as a TCP server inside the container.

   docker exec -it dartsim bash -c "cd /headless/dartsim && ./run.sh"

OR use the script:
   
   .\start-dartsim-tcp.ps1

NOTE: DARTSim must stay running for training to work. If you run it 
in the foreground, keep that terminal open. If you need it in background,
you'll need to run it differently or use screen/tmux inside container.
